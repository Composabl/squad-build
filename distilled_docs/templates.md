## 1. Log in

You must be logged in to the AMESA platform to access a template.

```bash
amesa login
```

## 2. Import components

#### Skill

```bash
amesa skill new
```

Can choose between:

- Controller
- Teacher
- Coach

#### Perceptor

```bash
amesa perceptor new
```

#### Agent

```bash
amesa agent new
```

Can choose between:

- Local target
- Docker target

#### Sim

```bash
amesa sim new [--docker]
```

#### Selector

amesa selector new
