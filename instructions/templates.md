### Creating a Controller or Learned Skill

```bash
AMESA login
composabl skill new
```

Select the desired component when prompted

### Creating a New Perceptor

Use the CLI to generate a perceptor template:

```bash
composabl perceptor new
```

This creates a directory structure:

```
perceptor_name/
├── perceptor_name/
│   ├── __init__.py
│   └── perceptor.py
├── pyproject.toml
└── README.md
```
